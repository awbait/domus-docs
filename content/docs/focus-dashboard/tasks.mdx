## Ideas

- [ ] Модуль: Погода #module #backlog
	Open-Meteo (бесплатный, без ключа). Виджет: температура + иконка + 5 дней. Расширенный: почасовой график, ощущается как, давление. [[focus-dashboard#Погода]]
- [ ] Модуль: Календарь #module #backlog
	CalDAV (Nextcloud) / Google Calendar / Яндекс. Виджет: ближайшие 3-5 событий. Расширенный: неделя/месяц. [[focus-dashboard#Календарь]]
- [ ] Модуль: Задачи / To-Do #module #backlog
	Простые списки. Интеграция с Todoist / TickTick или встроенный менеджер. Повторяющиеся задачи. [[focus-dashboard#Задачи / To-Do]]
- [ ] Модуль: Таймер / Помодоро #module #backlog
	Несколько одновременных таймеров, пресеты (5/10/25 мин), уведомление по окончании. [[focus-dashboard#Таймер / Секундомер]]
- [ ] Модуль: Трекер привычек #module #backlog
	Ежедневные привычки, streak-цепочка, визуализация. [[focus-dashboard#Трекер привычек]]
- [ ] Модуль: Финансы #module #backlog
	Быстрое добавление расходов, категории, итоги за месяц. Опционально: Тинькофф API. [[focus-dashboard#Финансы]]
- [ ] Модуль: Медиа (Jellyfin) #module #backlog
	Текущий трек/фильм, управление воспроизведением через Jellyfin API. [[focus-dashboard#Медиа]]
- [ ] Модуль: Инвентарь + Список покупок #module #backlog
	Домашний инвентарь с целевым количеством, голосовое добавление через Домовой, список покупок на телефон. [[focus-dashboard#Инвентарь + Список покупок]]
- [ ] Модуль: Заметки (sticky notes) #module #backlog
	Быстрые заметки, семейные объявления. [[focus-dashboard#Заметки]]
- [ ] Модуль: Фото (Immich) #module #backlog
	Слайд-шоу семейных фото через Immich API. [[focus-dashboard#Фото]]
- [ ] OIDC интеграция #auth #backlog
	Authentik / Authelia / Keycloak. Маппинг групп → роли. JWKS-валидация JWT. [[focus-dashboard#OIDC]]
- [ ] Swagger / OpenAPI #arch #backlog
	`swaggo/swag` аннотации в хэндлерах ядра и модулей → единый spec на `/api/swagger/`. [[focus-dashboard#Backend]]
- [ ] Кастомизация: акцентный цвет + фон #ui #backlog
	Per-user CSS variable `--primary`, фоновое изображение (URL или загрузка). [[focus-dashboard#Frontend]]
- [ ] Offline-режим (PWA) #pwa #backlog
	Service worker кэширует последние данные, дашборд работает без сети. [[focus-dashboard#Требования к мобильному режиму]]


## Planned

- [ ] Dockerfile multi-stage + docker-compose #infra #p1
	`Node build → Go build → runtime`. Frontend встроен в Go-бинарник через `embed`. `docker-compose.yml` для prod-деплоя рядом с Домовым.
- [ ] pill-tracker: push-уведомления (VAPID) #module #p1
	VAPID-ключи, хранение подписок браузеров (таблица `push_subscriptions` готова). Отправка: за 5 мин до дозы, +30 мин если не отмечена. Endpoint `POST /api/modules/pill-tracker/push/subscribe`. [[focus-dashboard#Уведомления]]
- [ ] Модуль: Управление Домовым (domovoy-control) #module #p2
	Backend: proxy к HTTP API Домового (текстовые команды, история, hub/spoke статус). Frontend: виджет 2x2 + полноэкранный режим с логом команд и статусом нод. Зависит от наличия HTTP API в Домовом. [[focus-dashboard#Модуль: Управление Домовым (MVP #2)]]
- [ ] Mobile layout mode #ui #p2
	`mode=mobile` в layout API. Нижняя навигация вместо боковой. Touch targets 48px. Адаптивная сетка 4 колонки.
- [ ] Wall: таймаут + возврат в shared #ui #p3
	`wall_timeout_minutes` из настроек. После N минут бездействия — автовозврат в shared-режим (без личных виджетов). [[focus-dashboard#Настенный монитор — переключение пользователя]]
- [ ] Wall: переключение пользователя + PIN #auth #p3
	Аватарки пользователей в углу wall-экрана. Тап → ввод 4-значного PIN → личные виджеты. Хранение PIN как bcrypt в таблице users. [[focus-dashboard#Настенный монитор — переключение пользователя]]

## Done

- [x] Backend Фаза 1: фундамент #arch
	Config (caarlos0/env + godotenv), DB (SQLite WAL + migrations runner), Module interface + Registry (MigrateAll, MountAll, HealthAll), chi-роутер + graceful shutdown.
- [x] Backend Фаза 1: auth #auth
	Модель, репозиторий, сервис (bcrypt + JWT), middleware (Bearer + cookie), handler. Setup wizard (первый запуск → `/setup`). Роли: admin / user / viewer.
- [x] Backend Фаза 1: pill-tracker модуль #module
	Полный fullstack-модуль: patients, medications, schedules, dose logs. CRUD API, статус доз на сегодня, история. Migrations embed.
- [x] Backend Фаза 2: layout #layout
	`GET/PUT /api/layout?mode=desktop|mobile` — per-user. `GET/PUT /api/admin/layout?mode=wall&shared=true` — shared wall (user_id = NULL). Исправлен баг SQLite UNIQUE с NULL: ручной UPDATE+INSERT вместо ON CONFLICT.
- [x] Backend Фаза 2: settings, links, backup, websocket #arch
	`GET/PATCH /api/admin/settings`, `CRUD /api/admin/links`, `GET /api/admin/backup` + `POST /api/admin/backup/restore`, gorilla/websocket Hub с Broadcast/BroadcastTo.
- [x] Frontend: auth + роутинг #ui
	LoginPage, SetupPage, AuthContext, защищённые маршруты, authApi.
- [x] Frontend: DashboardPage #ui
	Grid layout (react-grid-layout), edit mode, add/remove widgets, widget registry. Сохранение layout per-user. Исправлены: spurious saves (editModeRef guard), кнопка удаления (draggableCancel .nodrag), мигание виджетов (ждём modules перед рендером), автоочистка layout при отключении модуля.
- [x] Frontend: AdminPage #ui
	Sidebar навигация. Панели: пользователи, внешние ссылки (CRUD), настройки, модули (all / installed). Toggle enable/disable модуля, настройки модуля в диалоге.
- [x] Frontend: pill-tracker Widget + SettingsPage #module
	Виджет с дозами на сегодня, отметка дозы одним тапом. Страница настроек: patients, medications, schedules.
- [x] Wall mode #layout
	WallPage (`/wall`): full-screen, загружает shared layout. Admin: overlay-контролы (opacity-0 → hover), edit/done, add/remove widgets. Сохранение через `PUT /api/admin/layout`. Навигация: Monitor-иконка в header дашборда, Wall-макет в sidebar админки.
- [x] PWA: manifest.json #pwa
	`frontend/public/manifest.json` — name, icons, display standalone, theme_color.
